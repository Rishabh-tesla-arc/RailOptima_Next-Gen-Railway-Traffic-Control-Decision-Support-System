# RailOptima üöÇ

**Next-Gen Railway Traffic Control & Decision Support System**

RailOptima is an advanced, AI-powered web application designed to assist Indian Railways in managing complex traffic scenarios. It provides real-time monitoring, conflict detection, and intelligent optimization strategies to enhance efficiency, punctuality, and safety.

Link - https://indianrailoptima.netlify.app/

## üöÄ Key Features

- **Live Network Dashboard**: Real-time visualization of the railway network with interactive station nodes, status indicators, and weather alerts.
- **AI Conflict Resolution**: Automated detection of scheduling conflicts (e.g., track convergence) with AI-suggested rerouting solutions.
- **Traffic Simulation**: Interactive demos showcasing how the system handles high-priority trains (Rajdhani, Vande Bharat) vs. freight traffic.
- **Comprehensive Analytics**: KPI dashboards for network health, punctuality, and resource utilization.
- **Modern UI/UX**: A responsive, bilingual (English/Hindi) interface built with modern web technologies.

## üíª Getting Started

Follow these steps to set up the project locally:

1.  **Clone the repository**
    ```bash
    git clone https://github.com/Rishabh-tesla-arc/RailOptima_Next-Gen-Railway-Traffic-Control-Decision-Support-System.git
    cd RailOptima_Next-Gen-Railway-Traffic-Control-Decision-Support-System
    ```

2.  **Install dependencies**
    ```bash
    npm install
    ```

3.  **Run the development server**
    ```bash
    npm run dev
    ```

4.  **Open in Browser**
    Visit `http://localhost:5173` to view the application.
